Description

Your calculator can do the basic, but let's now increment it with some special use cases

Objectives
  1. Format numbers so that integer numbers are displayed on displayEditText hint without decimal part
  2. Give support for repeating an operation with the result of last operation by clicking equal again
    • the result of the previous operation should be the first term of the new operation;
    • the operator should remain the same;
    • the second term should remain the same;
    • ex: `3 * 2 = =` results on 12;
  3. If the second term of the operation is not set and equal is pressed make the operation with the first term on both sides of the operation
    • ex: `5 * =` results on 25
