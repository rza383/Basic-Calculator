Description

Once we have the layout, let's continue with our input. So far, the buttons are plain pictures. They don't seem to work. Infuse life into them by adding operations and showing the numbers you've typed on the screen.

Objectives

Power up the buttons you have created in Stage 1.

  1. After clicking on any number button, a corresponding number should be added to displayEditText text. Make sure you do not overwrite the number you've typed in earlier.
  2. A number displayed on displayEditText should not have extra zeroes to the left
  3. A number displayed on displayEditText should not have more than one dot
  4. A number with a decimal part should always have an integer part
  5. Clicking clearButton button set displayEditText back to initial state, that is, set text to "" and set hint to "0"
